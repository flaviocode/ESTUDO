ATALHOS
ALT + Z = quebra as linhas 
ALT + SHIFT + pra baixo = duplicar a dinha
CONTROL + COMMAND + 2 = divide a tela em duas
COMAND + [ ou ] = também serve como tabulação, porem melhor
COMAND + D = selecionar palavras na sequencia para trocar o texto
